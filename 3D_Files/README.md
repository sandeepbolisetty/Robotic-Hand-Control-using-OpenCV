# 3D Files

This folder contains all the 3D printable components required for the Gesture Control Robotic Hand project.

## Included Parts

| File Name | Description |
|------------|------------|
| files-e5p1 | Finger component |
| files-hand5 | Main hand assembly |
| files-servohorn | Servo horn adapter |
| fingerdesign-all-1 | Finger design component |
| forarm-9-2 | Forearm section |
| forarm-9-3 | Forearm section |
| forearm9-1 | Forearm section |
| hand5thumb3 | Thumb component |
| myminifactory-handback | Back cover of hand |
| servo-hold | Servo mounting bracket |
| servo-hold-1 | Servo mounting bracket |

## Printing Recommendations

- Material: PLA or PETG
- Layer Height: 0.2 mm
- Infill: 20–30%
- Supports: As required
- Nozzle Diameter: 0.4 mm

## Assembly Notes

1. Print all parts at 100% scale.
2. Remove supports and clean the printed parts.
3. Install servo motors into the servo holders.
4. Assemble fingers using joints and linkage mechanisms.
5. Connect tendons/strings to the servo horns.
6. Mount the hand assembly onto the forearm section.

## Project

These STL files are part of the **Gesture Control Robotic Hand using OpenCV** project, where hand gestures detected through computer vision are translated into robotic hand movements using Arduino and servo motors.
